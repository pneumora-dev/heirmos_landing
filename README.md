# Heirmos — Landing

[![smithery badge](https://smithery.ai/badge/heirmos/memory)](https://smithery.ai/servers/heirmos/memory)

[heirmos.com](https://heirmos.com) 랜딩 페이지. **Heirmos**는 여러 AI(Claude · ChatGPT · Grok · Gemini)가 공유하는 클라우드 기억 인프라입니다 — 어떤 AI와 대화하든 쌓아온 기억을 그대로 이어갑니다.

- 웹: <https://heirmos.com>
- 대시보드: <https://dashboard.heirmos.com>
- 이용 가이드: <https://dashboard.heirmos.com/guide>
- MCP 서버: `https://mcp.heirmos.com/mcp` — [Smithery에서 보기](https://smithery.ai/servers/heirmos/memory)

## Stack

Next.js 16 (App Router) + Tailwind 4 정적 사이트. Vercel 배포.

```bash
npm install
npm run dev      # http://localhost:3000
npm run build
```
